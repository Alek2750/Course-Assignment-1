Project Description

There is no business model involved with this CA.
Your task will be to produce a number of HTML pages with the sole purpose of “learning” while doing.
They all include some JavaScript + DOM manipulation running on the client and a few also include some simple server-interaction (AJAX)
When completed, your pages must be hosted on a Tomcat Server deployed under your own domain name and only accessible via HTTPS.
